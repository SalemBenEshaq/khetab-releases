# Khetab - AI-Powered Bilingual Speech Improvement App

Khetab is a mobile application that helps users improve their speaking skills 
in Arabic and English using AI. It provides detailed feedback on pronunciation, 
fluency, tone, and pacing.

## Download

👉 [Download Latest APK](https://github.com/SalemBenEshaq/khetab-releases/releases/tag/v1.0.0)

## Requirements

- Android device
- Internet connection
- Contact the author for backend server access

## Installation

1. Download the APK file from the link above
2. Open the APK on your Android device
3. If prompted, enable **Install from unknown sources**
4. Tap Install and wait for it to complete
5. Open Khetab and create your account

## Features

- 🎤 AI speech analysis (Arabic & English)
- 📊 Scores for Pronunciation, Fluency, Tone, and Pacing
- 📝 Speech transcription and text enhancement
- 📈 Progress tracking and session history
- 🏆 35 achievements and badge system
- ☁️ Cloud storage with Firebase
